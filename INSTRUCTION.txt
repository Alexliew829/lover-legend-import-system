Lover Legend 进口成本与库存系统 — 正式版 V2.5

部署步骤：
1. 上传 Frontend 文件夹内全部内容到 GitHub Repository 根目录并覆盖旧文件。
2. 等待 GitHub Pages 完成更新。
3. 打开正式网址并按 Ctrl + F5；已安装的 PWA 请完全关闭后重新打开。

Apps Script：
- Code.gs 没有修改。
- appsscript.json 没有修改。
- 不需要重新部署 Apps Script。
- 继续使用原本的 Web App URL。

V2.5 库存规则：
- 原进口数量属于只读历史，保持不变。
- 产品进口页载入旧进口编号后，数量代表当前剩余库存，可以增加或减少。
- 销售时减少，顾客退货或盘点修正时可以增加。
- 库存调整只改变当前库存及库存成本总值。
- Average Cost、原进口成本、海外运费比例及整批历史总成本保持不变。
